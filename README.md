# computability_exp

This repository contains two standalone HTML assignments for a computability theory course. Each page is designed to be opened directly in a browser and explored without any build step or external dependencies.

## Contents

- [assign_1.html](assign_1.html) - an interactive explanation of the Post Correspondence Problem and why it is undecidable.
- [assign_3.html](assign_3.html) - a visual explanation of regex equivalence, PSPACE, and EXPSPACE.
- [Exam Style Guide - BCS_402.pdf](Exam%20Style%20Guide%20-%20BCS_402.pdf) - reference material included in the repository.

## How To Use

Open any of the HTML files directly in a web browser. For example, double-click the file in your file explorer or drag it into an open browser window.

## Notes

- The repository is static; there is no install or build process.
- If you add new assignments later, list them here so the README stays current.
